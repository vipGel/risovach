## Getting Started
While developing run:
```bash
flutter pub run build_runner watch --delete-conflicting-outputs 
```

Before building apk run:
```bash
flutter pub get
```

```bash
flutter pub run build_runner build --delete-conflicting-outputs 
```

To build apk run:
```bash
flutter build apk --release 
```
